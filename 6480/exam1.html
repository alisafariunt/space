Exam Blueprint: What Earns Points

توضیح فارسی: تو امتحان، چی نمره می‌آورد؟

تو سبک امتحان پارسال، فقط "عدد" مهم نبود. باید بتونی بگی چرا این روش را انتخاب کردی و نتیجه یعنی چه. خیلی‌ها کد می‌زنند، ولی نمره را از تفسیر می‌گیرند یا از دست می‌دهند.

سوال‌ها معمولا چند مدل هستند. بعضی‌ها دام True/False هستند. بعضی‌ها محاسبه کوتاه دارند (مثل Tukey critical difference یا effect size). بعضی‌ها هم نوشتن دارند: "چه نتیجه‌ای گزارش می‌کنی و چرا".

اگر بخواهم ساده بگویم، جواب قوی سه قسمت دارد: روش، عدد کلیدی، تفسیر. اگر فقط یکی از این‌ها باشد، جواب نصفه می‌شود.

A strong answer almost always has these three elements:

  • Method choice: name the method and one reason why it fits the question
  • One key computation/output: the number that drives the decision
  • Plain interpretation: what it means for the research question

Fast scoring rule

If you can explain your reasoning in 2 to 4 sentences without reading software output, you are probably ready.

One-Page Formula Sheet (Exam 1)

توضیح فارسی: برگه فرمول یک صفحه‌ای

این قسمت را می‌تونی جدا پرینت بگیری و شب قبل امتحان سریع مرور کنی. هدفش این است که سر جلسه اگر فرمول یادت رفت، حداقل منطق و شکل کلی را بلد باشی و بتوانی مراحل را بنویسی (برای نمره جزئی هم خیلی کمک می‌کند).

این فرمول‌ها مخصوص همان چیزهایی است که تو امتحان پارسال دیدیم: one-way ANOVA، effect size، Tukey، contrasts، و power.

This is a quick formula sheet for the exact topics that appeared in the graded Exam 1 (2025).

One-way ANOVA
df1 = p - 1
df2 = N - p
F = MSA / MSE

Multiple comparisons
m (pairwise count) = p*(p-1)/2
Bonferroni alpha* = alpha / m

Effect size from ANOVA table
eta^2 = SS_A / SS_T
omega^2 = (SS_A - df1*MSE) / (SS_T + MSE)
omega^2 = max(0, omega^2)

Effect size conversions
f = sqrt( eta^2 / (1 - eta^2) )
f = sqrt( omega^2 / (1 - omega^2) )

Effect size for power (balanced one-way, exam-style)
f = sqrt( (df1 * F) / N )

Tukey critical difference (equal n)
CD = q_{alpha}(p, df_error) * sqrt(MSE / n)

Contrasts
Valid contrast: sum(c_j) = 0
Orthogonal (equal n): sum(cA_j * cB_j) = 0

Common Number Mistakes (Quick Fixes)

توضیح فارسی: اشتباه‌های عددی رایج (که نمره را می‌سوزاند)

خیلی وقت‌ها تو امتحان، مسئله سخت نیست. اشتباه‌های ریز عددی باعث می‌شود جواب غلط شود. این قسمت را چند بار مرور کن چون واقعا نمره‌اش راحت است.

SCM1

If omega^2 becomes negative in your computation, what do you report?

EN: Report omega^2 = 0. Negative omega^2 is truncated to 0 (do not report a negative effect size).

FA: باید omega^2 را 0 گزارش کنی. اگر منفی شد، truncate می‌شود و منفی گزارش نمی‌شود.

SCM2

For Tukey critical difference, what is p in the q-table lookup?

EN: p is the number of groups (treatments), not a degrees of freedom number.

FA: p یعنی تعداد گروه‌ها، نه df.

SCM3

In mediation, if indirect/total ratio is bigger than 1, is that an error?

EN: No. It can happen under competitive mediation when direct and indirect have opposite signs and partially cancel.

FA: نه. این در competitive mediation طبیعی است وقتی direct و indirect علامت مخالف دارند و همدیگر را cancel می‌کنند.

Memory line

EN: If a number looks "weird", ask: "Is this a known trap?" Many times, yes.

FA: اگر یک عدد عجیب به نظر می‌آید، اول فکر کن "دام امتحانی است یا اشتباه من؟" خیلی وقت‌ها دام است.

Partial Credit Strategy (If You Get Stuck)

توضیح فارسی: اگر گیر کردی، برای نمره جزئی چی بنویس؟

تو همین امتحان پارسال خود برگه نوشته بود "Partial credit is given" و گفته بود اگر نتوانستی حل کنی، بنویس چطور حل می‌کنی. این یعنی اگر تو مراحل را درست بنویسی، حتی اگر عدد آخر را اشتباه دربیاوری، باز نمره می‌گیری.

پس اگر گیر کردی، به جای خالی گذاشتن، این‌ها را بنویس: مدل چی هست، H0/H1 چی هست، از چه خروجی‌هایی چه چیزی برمی‌دارم (مثل MSE، df، N)، فرمول چی هست، و آخرش یک جمله تفسیر که اگر p<alpha بود چه نتیجه‌ای می‌گرفتم.

Template you can copy (EN)

1) State the model and define groups/variables. 2) Write H0/H1. 3) Identify what numbers you need from output (F, df, MSE, n). 4) Write the formula and show substitution. 5) Write one sentence interpretation tied to alpha.

قالب (FA): 1) مدل را بنویس و متغیرها را تعریف کن. 2) H0/H1 را بنویس. 3) بگو از خروجی کدام عددها را لازم داری (F، df، MSE، n). 4) فرمول را بنویس و جایگذاری کن. 5) یک جمله تفسیر که به alpha وصل باشد.

Decision Flowcharts (Fast Decision Rules)

توضیح فارسی: چند تا قانون تصمیم سریع که تو امتحان نجاتت می‌دهد

تو امتحان پارسال، سوال‌ها طوری بود که اگر فقط "خروجی نرم‌افزار" را نگاه کنی، ممکن است ندانی چی باید گزارش کنی. نمره اصلی معمولا برای این است که تو روش درست را انتخاب کنی و بعد یک جمله تفسیر درست بنویسی.

این flowchartها برای همین است. مثل یک نقشه سریع است. اگر وسط سوال گیر کردی، یک لحظه برگرد به این قسمت و ببین مسیر درست کدام است. بعد جواب را خیلی ساده و روشن بنویس.

On this exam style, most points come from good decision making. The math is usually short. These are the same decision rules I use when I answer questions fast.

Flow 1: One-way group comparison (ANOVA family)

Start: compare group means

1) What is the goal?
- "All pairwise comparisons" -> Tukey
- "Only a few planned comparisons" -> planned contrasts + Holm/Bonferroni
- "Every group vs a control" -> Dunnett (if allowed)

2) Are assumptions OK?
- Levene p > alpha AND residuals look OK -> standard one-way ANOVA (GLM)
- Levene p < alpha -> Welch ANOVA + Games-Howell (pairwise)

3) Is Y positive and right-skewed? (time, money, wait time)
- Yes -> Gamma GLM (GENMOD-style) with log link
- Also run a nonparametric method (Kruskal-Wallis + DSCF) and compare results

Flow 2: Mediation (what counts as "mediation exists")

Key test: bootstrap CI for indirect effect (a*b)

CI excludes 0 -> indirect effect is significant -> mediation exists

Then classify pattern:
- Indirect sig, direct (c') not sig -> full mediation
- Indirect sig AND direct sig, same sign -> partial mediation
- Indirect sig AND direct sig, opposite signs -> competitive mediation
  (Total effect can be near 0 and non-significant. That is not an error.)

Flow 3: Moderation (interaction test)

Test the interaction term X*W

If p < alpha -> moderation supported
If p >= alpha -> do NOT claim moderation

Exam tip: always write the alpha you used (0.05 or 0.01)

One sentence you can reuse

EN: "Based on the goal of the question and the assumption checks, I used [METHOD], and I report significance using alpha = [VALUE]."

FA: "با توجه به هدف سوال و چک کردن فرض‌ها، من از [روش] استفاده کردم و معنی‌داری را با alpha = [عدد] گزارش می‌کنم."

True/False Trap Zone (10 Core Traps from Exam 2025)

توضیح فارسی: 10 دام مهم True/False که تو امتحان پارسال بود

این بخش را واقعا باید قوی ببندی، چون سوال‌ها خیلی ریز هستند و اگر کلمات را دقیق نخونی راحت غلط می‌زنی. نکته‌اش این است که معمولا هر جمله یک دام دارد: "always"، "must"، "at least one"، یا قاطی کردن Type I و Type II.

من هر سوال را با جواب و دلیل کوتاه نوشتم. از خودت بخواه که قبل از نگاه کردن به جواب، 10 تا را سریع جواب بدهی و بعد دلیل خودت را با دلیل اینجا مقایسه کنی.

Q1

Statement: Cohen's effect size is always larger than omega-square for a treatment in a one-way ANOVA.

Answer: True (in this course, "Cohen's effect size" for one-way ANOVA usually means Cohen's f).

Why (EN): If you compute f from omega^2, then f = sqrt(omega^2 / (1 - omega^2)). For any 0 < omega^2 < 1, f is bigger than omega^2 (because you take a square root and also divide by something smaller than 1).

چرا (FA): تو این درس وقتی می‌گوین Cohen effect size برای ANOVA، معمولا منظور Cohen's f است. اگر f را از omega^2 بسازی، f همیشه از omega^2 بزرگ‌تر می‌شود (به جز حالت صفر). پس جمله درست است.

Q2

Statement: If an experimental design is orthogonal, then the p-values of the Type I and Type II sum of squares do not always have to be the same.

Answer: False.

Why (EN): In an orthogonal (balanced) design, the different SS "types" usually match for the main effects, so the p-values match. Differences show up mainly when the design is not orthogonal (imbalance, correlation between predictors, etc.).

چرا (FA): اگر طرح orthogonal و متعادل باشد، Type I و Type II (و اغلب Type III) برای اثرهای اصلی معمولا یکی می‌شوند. اختلاف‌ها بیشتر وقتی است که طرح نامتعادل یا غیرمتعامد باشد.

Q3

Statement: In a one-way ANOVA model in which the medians are not equal, but the means of the treatment levels are exactly equal, the parametric one-way ANOVA may sometimes reject the null hypothesis.

Answer: False (for the exam logic).

Why (EN): One-way ANOVA is about means, not medians. Different medians alone do not force rejection. If the group means are exactly equal in the sample, then SS_between = 0, so F = 0 and you cannot reject.

چرا (FA): ANOVA برای میانگین‌هاست نه median. فقط اینکه medianها فرق کنند، به تنهایی باعث رد شدن H0 نمی‌شود. اگر میانگین‌ها در نمونه واقعا دقیقا برابر باشد، SS_between صفر می‌شود و دیگر چیزی برای رد کردن نداریم.

Q4

Statement: One-way ANOVA has 4 levels. The overall model is significant at 5%. Using Bonferroni for all pairwise comparisons at 5%, at least one pairwise difference will be significant.

Answer: False.

Why (EN): A significant omnibus F does not guarantee any specific pairwise comparison will survive a strict correction like Bonferroni. The overall test can be significant while all corrected pairwise p-values are above 0.05.

چرا (FA): معنی دار شدن آزمون کلی ANOVA تضمین نمی‌کند که حتما یک مقایسه جفتی بعد از Bonferroni هم معنی دار شود. ممکن است اختلاف‌ها پخش باشند و هیچ جفتی بعد از اصلاح معنی دار نشود.

Q5

Statement: If MS_Treatment = MS_Error, then the estimate of the effect size is zero.

Answer: True.

Why (EN): If MS_Treatment = MS_Error, then F = 1. Most effect size estimates like omega^2 become 0 (and are often truncated to 0 when the formula gives a negative number).

چرا (FA): وقتی MS_treat و MS_error برابر شوند، F می‌شود 1. در این حالت اثر بین گروهی اضافه‌ای نسبت به خطا نداریم و omega^2 عملا صفر می‌شود.

Q6

Statement: In a 3-level treatment coded with two dummy variables, if the two dummy betas are equal, then the means of all three levels are equal.

Answer: False.

Why (EN): With dummy coding, beta1 and beta2 are differences versus the reference mean. If beta1 = beta2, it only implies groups 2 and 3 are equally far from the reference. The three means are equal only if beta1 = beta2 = 0.

چرا (FA): تو dummy coding، betaها اختلاف نسبت به گروه مرجع هستند. اگر beta1=beta2، فقط یعنی گروه 2 و 3 نسبت به مرجع به یک اندازه فرق دارند. برابری هر سه میانگین وقتی است که هر دو beta صفر باشند.

Q7

Statement: If omega^2 = 0 for the treatment, then the F statistic must be 0.

Answer: False.

Why (EN): F is a ratio of mean squares, so it is not 0 in general. In many formulas, omega^2 = 0 happens when F <= 1 (especially after truncating negative values to 0). A common case is F = 1, not F = 0.

چرا (FA): F نسبت دو MS است و معمولا صفر نمی‌شود. omega^2=0 معمولا وقتی رخ می‌دهد که F خیلی کوچک باشد و مخصوصا وقتی F=1 (یا کمتر از 1 و بعد truncation). پس جمله غلط است.

Q8

Statement: Holm's multiple comparison test for pairwise means controls the Type II familywise error rate.

Answer: False.

Why (EN): Holm controls familywise Type I error (false positives). Type II error is about false negatives, and Holm is not designed to control that as a familywise rate.

چرا (FA): Holm برای کنترل خطای نوع اول خانواده‌ای است (FWER). نوع دوم یعنی missed detection و این روش برای آن طراحی نشده.

Q9

Statement: If in each group the median equals the mean, then Brown-Forsythe and SPSS Levene test will be exactly equal.

Answer: True.

Why (EN): Levene uses deviations from group means. Brown-Forsythe uses deviations from group medians. If mean = median in each group, those deviations are the same, so the tests match.

چرا (FA): Levene فاصله از میانگین را می‌گیرد، Brown-Forsythe فاصله از median. اگر در هر گروه mean=median باشد، عملا همان فاصله‌ها می‌شود و نتیجه برابر می‌شود.

Q10

Statement: In mediation, if indirect effect is significant and positive but direct effect is significant and negative, you should consider an omitted mediator.

Answer: True.

Why (EN): This is a classic "inconsistent" or "competitive" pattern where the direct and indirect paths have opposite signs. It can be a sign the model is missing another mechanism (another mediator) or there is suppression.

چرا (FA): وقتی اثر مستقیم و غیرمستقیم علامت مخالف دارند، مدل "inconsistent/competitive mediation" می‌شود. این معمولا یعنی مکانیزم دیگری هم هست که در مدل نیومده و باید به omitted mediator یا suppression شک کرد.

One-Way ANOVA as Regression (Dummy Variables)

توضیح فارسی: ANOVA را مثل رگرسیون با dummy variables ببین

یک دام مهم امتحان این است که استاد ANOVA را به شکل رگرسیون می‌نویسد و ازت می‌خواهد تفسیر betaها را بگویی. اگر این را خوب بلد باشی، چند سوال پشت سر هم را راحت می‌گیری.

در مدل effects (Y = mu + alpha_j + error)، معمولا mu را می‌توانیم "میانگین کل" بگیریم، البته وقتی محدودیت‌های استاندارد مثل sum(alpha)=0 را داشته باشیم. در مدل dummy، beta0 معمولا میانگین گروه مرجع است. beta1 و beta2 اختلاف گروه‌های دیگر نسبت به مرجع هستند.

دام معروف همان سوال True/False بود: اگر beta1=beta2، یعنی گروه 2 و 3 نسبت به مرجع به یک اندازه اختلاف دارند، ولی لزوما یعنی هر سه گروه برابرند؟ نه. برای برابری هر سه میانگین باید هر دو beta صفر باشند.

In Exam 2025, a core skill was translating a one-way ANOVA into regression with dummy variables and then writing hypotheses in beta language.

Two equivalent models

Effects model:
	Y_ij = mu + alpha_j + epsilon_{i(j)}

Dummy regression model (3 levels):
Y_ij = beta0 + beta1 X1 + beta2 X2 + epsilon_ij

Dummy coding table (3 groups)

This is the quickest way to not get confused on the exam. Pick a reference group (often level 1). Then define dummies like this:

Group X1 X2
Group 1 (reference) 0 0
Group 2 1 0
Group 3 0 1

فارسی: تو امتحان اگر جدول را تو ذهنت داشته باشی، تفسیر betaها هم راحت می‌شود. گروه مرجع هر دو dummy اش صفر است.

How betas map to group means

mu_1 = beta0
mu_2 = beta0 + beta1
mu_3 = beta0 + beta2

So:

  • beta1 = mu2 - mu1
  • beta2 = mu3 - mu1

فارسی: اگر beta1 = beta2 باشد، فقط یعنی (mu2 - mu1) = (mu3 - mu1) و نتیجه می‌شود mu2 = mu3. ولی این هنوز یعنی هر سه تا برابرند؟ نه. برای برابری هر سه، باید هر دو beta صفر باشند.

What is mu in the effects model?

In the effects model, mu is the grand mean when we use the usual constraint sum(alpha_j) = 0. In a balanced design (equal n), that is the simple average of the group means. In an unbalanced design, it is a weighted average.

فارسی: mu تو مدل effects معمولا "میانگین کل" است (با محدودیت sum(alpha)=0). اگر nها برابر باشند، میانگین ساده میانگین‌های گروه‌ها می‌شود. اگر nها برابر نباشد، میانگین وزن‌دار می‌شود.

What does epsilon_{i(j)} mean?

epsilon is the random error term. The parentheses i(j) simply mean "the i-th observation inside group j" (nested indexing). On an exam, one clean sentence is enough.

فارسی: epsilon همان خطای تصادفی است. i(j) یعنی مشاهده iام داخل گروه jام. یعنی indexها تو در تو هستند، همین.

Interpretation

  • beta0 is the mean of the reference group (the group where X1=0 and X2=0).
  • beta1 is the mean difference between group 2 and the reference group.
  • beta2 is the mean difference between group 3 and the reference group.

Hypotheses for "equal means"

H0: beta1 = 0 AND beta2 = 0
	H1: at least one beta is not 0
Q11a

In the dummy regression model, what does beta0 represent? In the effects model, what does mu represent?

EN: In the dummy model, beta0 is the mean of the reference group. In the effects model, mu is the grand mean (under the usual constraint like sum(alpha_j)=0).

FA: تو مدل dummy، beta0 یعنی میانگین گروه مرجع. تو مدل effects، mu یعنی میانگین کل (با محدودیت‌های استاندارد مثل sum(alpha)=0).

Q11b

Write the null and alternative hypotheses for "equal means" using the dummy regression model with two dummy variables.

EN: H0: beta1 = 0 AND beta2 = 0. H1: beta1 != 0 OR beta2 != 0.

FA: H0: هر دو beta صفر هستند. H1: حداقل یکی از betaها صفر نیست.

Q11c

What does epsilon_{i(j)} mean? What does the parenthesis i(j) mean?

EN: epsilon is the random error. i(j) just means "the i-th observation within group j" (nested indexing).

FA: epsilon همان خطای تصادفی است. i(j) یعنی مشاهده iام داخل گروه jام (اندیس تو در تو).

Exam trap

beta1 = beta2 does not mean all three means are equal. It only means groups 2 and 3 are equally far from the reference.

# H0: beta1 = beta2 = 0 is the same as the overall F test for group

Optional: Two-Way (Factorial) ANOVA + Interaction

توضیح فارسی: اگر از factorial ANOVA و interaction سوال شد

امتحان پارسال بیشتر one-way بود، ولی بعضی وقت‌ها استاد یک سوال کوتاه از factorial ANOVA هم می‌پرسد. اگر این آمد، دامش معمولا "interaction" است.

interaction یعنی اثر عامل A به سطح عامل B بستگی دارد. اگر interaction معنی‌دار باشد، دیگه تفسیر main effect تنها خیلی قابل اعتماد نیست. باید ساده بگویی: "اثر A در سطح‌های مختلف B فرق می‌کند" و بعد از simple effects حرف بزنی.

Two-way ANOVA means you have two factors (A and B). The key idea is the interaction term.

Two-way ANOVA model:
Y_ijk = mu + A_i + B_j + (A*B)_ij + error
SC-TW1

If the interaction is significant, what do you report first?

EN: Report and interpret the interaction first. Then discuss simple effects (how A changes across levels of B, or how B changes across levels of A).

FA: اول interaction را گزارش و تفسیر کن. بعد درباره simple effects بنویس (اثر A در سطح‌های مختلف B، یا برعکس).

SC-TW2

If the interaction is NOT significant, what is the usual next step?

EN: You can interpret main effects (A and B). If a factor has more than 2 levels and is significant, use post hoc comparisons (like Tukey) for that factor.

FA: می‌توانی main effectها را تفسیر کنی (A و B). اگر یک عامل چند سطح داشته باشد و معنی‌دار شود، post hoc مناسب مثل Tukey را برای همان عامل اجرا کن.

Memory line

EN: Interaction significant = "it depends", so do not oversell main effects.

FA: interaction معنی‌دار = "بستگی دارد"، پس main effect تنها را زیادی قطعی تفسیر نکن.

Quick code (just for recognition)

# SAS
proc glm data=mydata;
  class A B;
  model Y = A B A*B;
run;
quit;

Unequal Variances + Unequal n: What Happens to Type I Error?

توضیح فارسی: وقتی واریانس‌ها برابر نیست و n هم برابر نیست، Type I error چه می‌شود؟

این سوال تو امتحان پارسال آمد. قانونش ساده است ولی خیلی‌ها قاطی می‌کنند. وقتی گروه با واریانس بزرگ‌تر نمونه کوچک‌تر داشته باشد، تست ANOVA معمولا "liberal" می‌شود یعنی احتمال خطای نوع اول بالا می‌رود. برعکس، اگر گروه با واریانس بزرگ‌تر نمونه بزرگ‌تر داشته باشد، تست "conservative" می‌شود یعنی احتمال رد کردن اشتباه کمتر می‌شود.

به زبان خیلی ساده، بدترین حالت این است که گروه شلوغ‌تر (variance بالا) را با n کوچک بگیری. چون برآورد واریانس ناپایدار می‌شود و F-test رفتار بدتری پیدا می‌کند.

اگر استاد بپرسد در این شرایط چه کار می‌کنیم، جواب خوب این است: Welch ANOVA یا روش‌های robust و بعد pairwise مناسب مثل Games-Howell. اگر داده skew باشد، گاهی transformation یا روش ناپارامتریک هم کمک می‌کند.

This is a classic exam concept: if the homogeneity of variance assumption is violated and sample sizes are unequal, the actual Type I error rate can move away from the nominal alpha.

Pattern What happens Memory Hook
Small n from high-variance groups Type I error increases (test becomes liberal) noisy + small = danger
Small n from low-variance groups Type I error decreases (test becomes conservative) noisy + big = safer

یادآوری سریع: noisy + small = danger (liberal). noisy + big = safer (conservative).

Q12a

When population variances are not homogeneous and sample sizes are unequal: if the smaller samples come from the more heterogeneous populations, what happens to the actual Type I error rate?

EN: Type I error increases. The test becomes liberal.

FA: خطای نوع اول بالا می‌رود و آزمون liberal می‌شود.

Q12b

Same setup: if the smaller samples come from the more homogeneous populations, what happens to the actual Type I error rate?

EN: Type I error decreases. The test becomes conservative.

FA: خطای نوع اول کم می‌شود و آزمون conservative می‌شود.

Why this happens (simple intuition)

ANOVA uses a pooled error estimate (MSE). When group sizes are unequal, the pooled error is influenced more by the large groups. So if the large groups happen to be the low-variance groups, the pooled MSE can look "too small" compared to the noisy small group. That can make the F test too aggressive and inflate Type I error.

فارسی: وقتی nها برابر نیست، MSE بیشتر تحت تاثیر گروه‌های بزرگ‌تر است. اگر گروه بزرگ‌تر واریانسش کم باشد، MSE کوچک‌تر دیده می‌شود و آزمون F راحت‌تر معنی‌دار می‌شود. برای همین Type I error بالا می‌رود (liberal).

What to write on the exam (2 sentences)

EN: "With heteroscedasticity and unequal sample sizes, the nominal 5% Type I error can change. If smaller groups come from higher-variance populations, the test becomes liberal (Type I increases), so I would use Welch ANOVA and a heteroscedastic-friendly pairwise method."

FA: "اگر واریانس‌ها برابر نباشد و nها هم برابر نباشد، احتمال خطای نوع اول می‌تواند از 5% منحرف شود. اگر گروه‌های کوچک‌تر از جامعه پرنوسان‌تر بیایند، آزمون liberal می‌شود (Type I بالا می‌رود)، پس Welch و مقایسه‌های مناسب‌تر لازم است."

What you do in practice

  • Use Welch ANOVA if variances are not equal.
  • Use Games-Howell style pairwise comparisons (works better under heteroscedasticity).
  • If the distribution is very skewed, consider a transformation or a nonparametric approach.

Effect Size from F (Fixed Effects) + Association for Random Effects

توضیح فارسی: محاسبه effect size از روی F (مثل سوال 13 امتحان)

سوال پارسال یک پاراگراف داد و فقط از روی F و dfها ازت effect size خواست. این مدل سوال خیلی محتمل است چون هم مفهوم را تست می‌کند هم محاسبه کوتاه را.

تو one-way ANOVA چند عدد مهم داریم: eta^2، omega^2 و Cohen's f. معمولا omega^2 نسبت به eta^2 کمتر خوش‌بینانه است. Cohen's f را هم می‌شود از eta^2 یا omega^2 ساخت و برای power هم استفاده می‌شود.

تو همین امتحان پارسال، با F(1,194)=9.08 و N=196 برای دو گروه، omega^2 حدود 0.0396 شد و Cohen's f حدود 0.203. این یعنی اثر کوچک تا متوسط (نزدیک کوچک).

برای random effects هم دنبال یک strength of association مثل ICC می‌گردیم. در سبک سوال پارسال، همان rho_I را از روی F و N ساختند و تفسیر مشابه دادند.

From an F test to omega squared

If you have a one-way ANOVA with p groups and total sample size N, and you know the omnibus F, a useful exam-ready formula is:

df1 = p - 1

omega^2 = (df1 * (F - 1)) / (df1 * (F - 1) + N)

# Same idea written another common way:
omega^2 = (df1 * (F - 1)) / (df1 * F + df2 + 1)

Cohen's f from omega squared (or eta squared)

f = sqrt( omega^2 / (1 - omega^2) )

# If you have eta^2 instead:
f = sqrt( eta^2 / (1 - eta^2) )

If eta squared is given (this happened in Exam 2025)

In the exam paragraph, they reported eta^2 = 0.045. You can compute Cohen's f directly (no extra steps): 

eta^2 = 0.045
f = sqrt(0.045 / (1 - 0.045)) = 0.217

فارسی: اگر eta^2 را خود سوال داده باشد، راحت‌ترین کار این است که مستقیم Cohen's f را از همان بسازی. این هم سریع است هم احتمال اشتباه کمتر دارد.

Quick eta squared from F and df (if eta^2 is NOT given)

eta^2 = (F * df1) / (F * df1 + df2)

Example: for F(1, 194) = 9.08:

eta^2 = (9.08*1) / (9.08*1 + 194) = 0.0447

Worked example (matches the 2025 exam style)

Given: F(1, 194) = 9.08
p = 2  => df1 = 1
N = 196

omega^2 = (1*(9.08 - 1)) / (1*(9.08 - 1) + 196)
        = 8.08 / 204.08
        = 0.0396

f = sqrt(0.0396 / (1 - 0.0396)) = 0.203

Interpretation (simple)

EN: omega^2 = 0.04 means about 4% of outcome variance is explained by the condition in this setting. f ≈ 0.20 is around small-to-medium by common benchmarks.

FA: omega^2 حدود 0.04 یعنی حدود 4 درصد از تغییرات Y با شرط آزمایش توضیح داده می‌شود. f حدود 0.20 معمولا کوچک تا متوسط حساب می‌شود.

Q13a

Fixed effects version (Exam 2025 style): given F(1,194)=9.08 and N=196, compute an effect size and interpret it.

EN: One clean way is to compute omega^2 then convert to Cohen's f. Here omega^2 ≈ 0.0396 and f ≈ 0.203, so the effect is small-to-medium.

FA: یک راه خوب این است که omega^2 را بسازی و بعد Cohen's f را حساب کنی. اینجا omega^2 ≈ 0.0396 و f ≈ 0.203 می‌شود، یعنی اثر کوچک تا متوسط.

Extra note: If the question already gives eta^2 = 0.045, you can also compute f = sqrt(eta^2/(1-eta^2)) ≈ 0.217. It tells the same story (small-to-medium).

Q13b

Random effects version (Exam 2025 style): compute a strength of association measure and interpret it.

EN: A simple exam-friendly measure is rho_I = (F - 1)/(F - 1 + N). Here rho_I ≈ 0.0396, meaning the association is weak (around 4%).

FA: یک معیار ساده برای امتحان این است: rho_I = (F - 1)/(F - 1 + N). اینجا rho_I ≈ 0.0396 است، یعنی ارتباط ضعیف است (حدود 4%).

نکته: تو این مثال ساده، عدد rho_I خیلی شبیه omega^2 می‌شود. ولی تفسیرشان فرق دارد: یکی effect size برای fixed effects، یکی strength of association برای random effects.

Random effects association (exam-friendly version)

For a simple one-way random effects interpretation, one exam-friendly association measure is:

rho_I = (F - 1) / (F - 1 + N)

This produces a number in the same 0 to 1 range. In plain language, it is the share of outcome variation that is "linked to the condition" in this setup (bigger means stronger association).

فارسی: این عدد بین 0 تا 1 است و می‌توانی بگویی "چه سهمی از تغییرات به شرط/گروه مربوط است". عدد بزرگ‌تر یعنی ارتباط قوی‌تر.

Tukey HSD: Reading Output + Critical Difference (Show Computation)

توضیح فارسی: Tukey تو امتحان خیلی مهم است (هم خوندن خروجی هم محاسبه)

دو مدل سوال Tukey تو امتحان پارسال بود: یکی اینکه یک جدول Tukey grouping داد و گفت با critical difference کدام جفت‌ها معنی دار هستند. دومی اینکه یک ANOVA table داد و گفت critical difference را حساب کن.

اشتباه رایج این است که فقط بزرگ‌ترین و کوچک‌ترین را نگاه کنند و سریع بگوین "همه فرق دارند". در حالی که باید تک تک اختلاف‌ها را با critical difference مقایسه کنی. تو همون امتحان، یکی از جفت‌ها دقیقا نزدیک مرز بود و اشتباه زیاد اتفاق افتاد.

Critical difference formula (equal n)

CD = q_{alpha}(p, df_error) * sqrt(MSE / n)

Where:

  • p = number of groups
  • df_error = error degrees of freedom
  • MSE = mean square error (within)
  • n = observations per group (equal n case)
  • q = studentized range critical value

Worked example (from Exam 2025 style)

SPSS ANOVA output shows: MSE = 7.545, df_error = 12, p = 3, total N = 15, so n = 5 per group. If q = 3.77: 

CD = 3.77 * sqrt(7.545/5)
  	   = 3.77 * 1.228
  	   = 4.62
Q15

Compute Tukey's critical difference. Given MSE = 7.545, n = 5, and q = 3.77, what is CD?

EN: CD = 3.77 * sqrt(7.545/5) = 4.62.

FA: CD = 3.77 * sqrt(7.545/5) = 4.62.

Exam detail

Always show the steps: identify MSE, identify n, look up q, then compute. "Just giving a number" often loses points.

How to decide significance (critical difference method)

For any pair of means |ybar_i - ybar_j|:

  • If the difference is greater than CD, the pair is significant.
  • If the difference is less than or equal to CD, the pair is not significant.

EN: This is the classic trap. People see "1.5 is basically 1.51" and mark it significant. But Tukey CD is a hard cutoff: if it is below CD, it is not significant.

FA: این دقیقا دام رایج است. بعضی‌ها می‌گوین "1.5 خیلی نزدیک 1.51 است، پس معنی‌دار". ولی قانون CD واضح است: اگر اختلاف از CD کمتر باشد، معنی‌دار نیست.

Tukey vs Scheffe: Why They Can Disagree

توضیح فارسی: چرا Tukey و Scheffe ممکن است نتیجه متفاوت بدهند؟

این هم تو امتحان پارسال بود. Tukey برای "همه مقایسه‌های جفتی" طراحی شده و معمولا نسبت به Scheffe توان بیشتری دارد. Scheffe محافظه‌کارتر است چون برای هر نوع contrast قابل استفاده است.

پس کاملا طبیعی است که یک جفت در Tukey معنی دار شود ولی در Scheffe مرزی یا غیرمعنی دار شود. اینجا باید تو تفسیر بنویسی که اختلاف وجود دارد ولی با روش محافظه‌کارتر تایید قوی نگرفت. همین جمله‌ها نمره می‌آورد.

In Exam 2025, there was a question asking to run both Tukey and Scheffe and explain differences. The key story is simple:

  • Tukey is designed for all pairwise mean comparisons, so it is efficient for that goal.
  • Scheffe protects against a wider class of comparisons (any contrast), so it is more conservative.

Mini example (matches the social media question)

In the graded example, Tukey reported p = 0.0416 for WebS2 vs WebS1 (significant at 5%), but Scheffe reported p = 0.0528 (not significant at 5%). That is not a contradiction. It just means the evidence is not strong enough under Scheffe's stricter protection.

فارسی: اینجا می‌تونی همزمان دو جمله بنویسی: "Tukey معنی‌دار گفت" و "Scheffe تایید نکرد چون محافظه‌کارتر است". همین می‌شود تفسیر درست و نمره‌دار.

One good exam sentence

EN: "Tukey suggests pair A vs B is significant, but Scheffe does not; this is expected because Scheffe is more conservative, so the evidence is weaker under stricter familywise protection."

FA: "Tukey یک تفاوت را معنی دار نشان داد ولی Scheffe تایید نکرد. این طبیعی است چون Scheffe محافظه‌کارتر است، پس شواهد تحت معیار سخت‌تر ضعیف‌تر می‌شود."

Orthogonal Contrasts (3 Groups) + How to Test Them

توضیح فارسی: کنتراست‌های متعامد (اینجا معمولا نمره از دست می‌رود)

تو امتحان پارسال یک سوال داشت که گفت دو تا کنتراست orthogonal بساز. دانشجو نمره از دست داد چون کنتراست‌ها orthogonal نبودند. پس این بخش را جدی بگیر.

برای 3 گروه، حداکثر 2 کنتراست orthogonal داریم. یک انتخاب ساده این است: C1 گروه 1 در برابر گروه 2، و C2 میانگین گروه 1 و 2 در برابر گروه 3. این دو تا هم جمع ضرایبشان صفر است و هم ضرب داخلی‌شان صفر می‌شود.

بعد از ساختن ضرایب، باید تفسیر کنی: هر کنتراست چه سوالی را می‌پرسد. و بعد با t-test (یا F تک درجه آزادی) معنی‌داری را گزارش کنی.

Two orthogonal contrasts for 3 groups (equal n)

Let group means be mu1, mu2, mu3. A clean orthogonal pair is:

C1: [  1, -1,  0 ]   # Group 1 vs Group 2
C2: [  1,  1, -2 ]   # Average(Group 1,2) vs Group 3

They are valid contrasts because coefficients sum to 0. They are orthogonal (equal n case) because:

1*1 + (-1)*1 + 0*(-2) = 0

How to test a contrast (formula)

psi_hat = sum(c_j * ybar_j)
SE(psi_hat) = sqrt( MSE * sum( c_j^2 / n_j ) )
t = psi_hat / SE(psi_hat)

R quick code (recommended for exam prep)

fit <- aov(Rating ~ Website, data = df)
emm <- emmeans::emmeans(fit, ~ Website)

contrast(emm, list(
  C1 = c( 1, -1,  0),
  C2 = c( 1,  1, -2)
))

Common mistake

People write two contrasts that each sum to 0, but they forget to check orthogonality (dot product = 0 for equal n). Both checks matter.

Power + Sample Size for One-Way ANOVA (5% vs 1%)

توضیح فارسی: power و sample size هم مثل پارسال می‌آید

تو امتحان پارسال ازت power در 5% و 1% خواست. نکته‌اش این است که وقتی alpha را سخت‌تر می‌کنی (مثلا 0.01)، power کم می‌شود. پس انتظار نداشته باشی power در 1% بالاتر باشد. برعکس است.

بعد هم سوال sample size برای power = 0.85 بود. اینجا دام بزرگ این است که جواب را رند نکرده یا روش را ننوشته باشی. باید نشان بدهی با چه روشی n را پیدا کردی (مثلا با pwr.anova.test) و آخرش هم رو به بالا گرد کنی.

یه نکته خیلی کلیدی که تو سوال پارسال لازم شد این بود که اول effect size (Cohen's f) را بسازی و بعد power را حساب کنی. یک روش سریع (سبک همون امتحان) این است: f = sqrt(((k-1)*F)/N).

Core relationships

  • Bigger N or bigger effect size f makes power higher.
  • Smaller alpha (like 0.01) makes power lower.
  • More noise (bigger variance) makes power lower.

R code for power (matches the 2025 exam style)

From the Social Media ANOVA example in Exam 2025: F(2,27) = 3.409, k = 3, n = 10 per group, so N = 30. A fast way to get Cohen's f is: 

f = sqrt(((k-1)*F)/N)
  = sqrt((2*3.409)/30)
  = 0.477
library(pwr)

k <- 3           # number of groups
n <- 10          # n per group

Serial Mediation (Model 6 Style) Quick Guide

توضیح فارسی: سریال مدیاسیون را ساده و امتحانی یاد بگیر

اگر تو امتحان ازت مدل سریال بخواهد، معمولا دنبال این است که بفهمی "چند تا مسیر غیرمستقیم داریم" و هر کدام چه معنایی دارد. مهم‌ترین دام هم این است که فکر کنی اگر total effect معنی‌دار نبود پس mediation نیست. این غلط است.

تو مدل سریال (مثل Hayes Model 6)، X می‌تواند هم از طریق M1 به Y برسد، هم از طریق M2، هم از طریق زنجیره M1 بعد M2. پس باید CI هر کدام را جدا چک کنی.

Serial mediation means the mediators have an order: X -> M1 -> M2 -> Y. In Hayes language, this is Model 6.

The 3 indirect paths (what you report)

Path a1: X -> M1
Path a2: X -> M2
Path d21: M1 -> M2
Path b1: M1 -> Y (controlling X and M2)
Path b2: M2 -> Y (controlling X and M1)

Indirect via M1 only:      ind_m1     = a1 * b1
Indirect via M2 only:      ind_m2     = a2 * b2
Serial (chain) indirect:   ind_serial = a1 * d21 * b2

Total indirect = ind_m1 + ind_m2 + ind_serial
Total effect c = direct c' + total indirect

How to decide significance (exam style)

EN: Use bootstrap CI for each indirect path. If the CI does not include 0, that indirect path is significant.

FA: برای هر مسیر غیرمستقیم جداگانه CI بوت‌استرپ را نگاه کن. اگر صفر داخل CI نباشد، آن مسیر معنی‌دار است.

SC-SM1

In serial mediation, how many specific indirect effects do you usually report?

EN: Three: ind_m1, ind_m2, and ind_serial.

FA: سه تا: مسیر از طریق M1، مسیر از طریق M2، و مسیر زنجیره‌ای (M1 بعد M2).

SC-SM2

If the serial indirect effect is negative and significant, what does that mean in plain language?

EN: The chain X -> M1 -> M2 -> Y carries a real effect that decreases Y overall. It often happens when one link in the chain is negative (for example b2 is negative).

FA: یعنی زنجیره X -> M1 -> M2 -> Y یک اثر واقعی دارد که در مجموع Y را کم می‌کند. معمولا وقتی یکی از لینک‌ها منفی باشد (مثلا b2) این اتفاق می‌افتد.

One good exam sentence (template)

EN: "The indirect effect through M1 is significant (bootstrap CI excludes 0), and the serial indirect effect is [positive/negative]. This suggests X influences Y partly through the ordered mechanism M1 then M2."

FA: "اثر غیرمستقیم از طریق M1 معنی‌دار است (CI صفر را شامل نمی‌شود) و اثر زنجیره‌ای [مثبت/منفی] است. یعنی X بخشی از اثرش را از طریق مکانیزم مرتب M1 بعد M2 به Y منتقل می‌کند."

Exam Drills (12 High-Value Q&A, Based on Real Exam Traps)

توضیح فارسی: تمرین‌های مهم (بدون نگاه کردن به جواب)

این 12 تا تمرین را من برای این گذاشتم که مثل امتحان، سریع تصمیم بگیری و درست جمله‌بندی کنی. لطفا اول سوال را بخوان و خودت 2 تا 4 جمله جواب بنویس. بعد جواب اینجا را چک کن.

اگر جایی گیر کردی، فقط همون یک جمله را بنویس که "اشتباه من چی بود". این کار ساده، واقعا باعث می‌شود دفعه بعد همون دام را تکرار نکنی.

SC1

Question: The indirect effect is 0.22, 99% CI [0.15, 0.31]. The total effect is 0.04, p = .40. Is mediation present? What type?

EN: Yes, mediation is present because the indirect effect is significant (the CI does not include 0). A non-significant total effect does not kill mediation. This pattern is often competitive mediation (direct and indirect have opposite signs), so an indirect/total ratio above 1 can happen.

FA: بله، mediation داریم چون CI اثر غیرمستقیم صفر را شامل نمی‌شود. غیرمعنی‌دار بودن total effect به معنای نبود mediation نیست. این حالت معمولا competitive mediation است (اثر مستقیم و غیرمستقیم علامت مخالف دارند)، پس نسبت indirect/total می‌تواند بالاتر از 1 باشد.

یک جمله نمره‌دار: "Indirect معنی‌دار است، پس mediation وجود دارد. Total غیرمعنی‌دار می‌تواند به خاطر cancel شدن direct و indirect باشد (competitive mediation)."

SC2

Question: Interaction term X × W: p = .03. Alpha is .01. Can you claim moderation?

EN: No. At alpha = 0.01, p = .03 is not significant, so moderation is not supported.

FA: نه. چون alpha = 0.01 است و p = 0.03 بزرگ‌تر است، interaction معنی‌دار نیست، پس moderation تایید نمی‌شود.

Exam sentence (EN): "The interaction was not significant at the 1% level (p = .03), so moderation is not supported."

جمله امتحانی (FA): "اثر تعاملی در سطح 1% معنی‌دار نبود، پس شواهد برای moderation نداریم."

SC3

Question: Hold-time residuals are right-skewed (skewness = 1.41). Why does this matter? What do you do?

EN: Right-skewed residuals are a normality concern for standard GLM ANOVA. Levene can still pass because Levene is about variance equality, not normality. So GLM is not "blocked", but skewness is a reason to also fit a Gamma model with log link (GENMOD style) and run a nonparametric method (DSCF). Then I compare conclusions and report stable vs method-sensitive pairs.

FA: کجی راست یعنی از نظر نرمال بودن residualها برای GLM مشکل داریم. ولی Levene ممکن است پاس شود چون Levene برای برابری واریانس‌هاست، نه نرمال بودن. پس GLM قابل اجرا هست، اما skewness باعث می‌شود Gamma با log link (مثل GENMOD) و nonparametric (DSCF) را هم اجرا کنم. بعد نتیجه‌ها را مقایسه می‌کنم و جفت‌های stable و method-sensitive را جدا گزارش می‌کنم.

نکته: اینجا هدف این نیست که بگویی "ANOVA غلط است". هدف این است که بگویی "برای robustness، مدل‌های مناسب skew هم اجرا می‌کنم".

SC4

Question: What is the difference between Tukey and Bonferroni?

EN: Tukey is built for all pairwise comparisons after ANOVA (studentized range) and is usually more powerful for that goal. Bonferroni is a general correction (alpha/m) and gets conservative fast as the number of tests grows. Use Tukey for all pairs. Use Bonferroni (or Holm) for a small number of planned tests.

FA: Tukey مخصوص مقایسه‌های جفتی بعد از ANOVA است و معمولا برای همین هدف توان بیشتری دارد. Bonferroni یک قانون کلی است (alpha تقسیم بر تعداد تست‌ها) و وقتی تست زیاد شود خیلی محافظه‌کار می‌شود. برای همه جفت‌ها Tukey بهتر است، برای چند فرضیه مشخص Bonferroni یا Holm.

SC5

Question: Problem 3 style: in a sequential planned-contrast rule (X3, X4, X6, X8, X10), C1 and C2 are significant but C3 is not. Which option does the retailer choose?

EN: Choose X6 ($6). The sequential rule requires stopping at the *first* non-significant contrast. Since C3 is not significant, the sequence stops immediately. C4 is never tested. We conclude X6 is the lowest amount not significantly worse than higher amounts.

FA: باید همان جایی که اولین بار non-significant می‌شود توقف کنی و همان گزینه را انتخاب کنی. چون از آن به بعد، تفاوت عملی/آماری با گزینه‌های بالاتر ثابت نشده و هزینه کمتر است.

یادآوری: این همون منطق "stop at the first non-significant contrast" است.

EN: Important detail: once C3 is not significant, you stop. That means C4 is not tested and it does not affect the decision. If someone asks, C4 is the next contrast (like X8 vs X10), but you never reach it under the rule.

FA: نکته مهم: وقتی C3 غیرمعنی‌دار شد، دیگه متوقف می‌شوی. یعنی C4 اصلا تست نمی‌شود و برای تصمیم انتخاب گزینه، دیگه نقشی ندارد. اگر سوال بپرسد، C4 کنتراست بعدی است (مثل X8 در برابر X10)، ولی طبق قانون ترتیبی به آن نمی‌رسیم.

SC6

Question: Why is a serial mediation model (like Model 2) preferred over running two single mediator models separately?

EN: Model 2 tests all indirect paths together, accounts for relationships between mediators, and allows formal contrast tests between indirect effects. Separate models can be biased because each ignores the other mediator.

FA: مدل سریال همه مسیرها را همزمان تست می‌کند، ارتباط بین mediatorها را کنترل می‌کند و امکان contrast بین مسیرهای غیرمستقیم را می‌دهد. مدل‌های جدا جدا می‌توانند bias داشته باشند چون هر کدام یکی از mediatorها را حذف می‌کند.

SC7

Question: Levene's test: F(3,377) = 2.16, p = .092. What does this mean?

EN: Not significant at 5%. We fail to reject equal variances, so the homogeneity assumption looks OK and standard ANOVA is fine (no need for Welch based on this test).

FA: در سطح 5% معنی‌دار نیست. پس فرض برابری واریانس‌ها رد نمی‌شود و ANOVA استاندارد قابل دفاع است (بر اساس این تست، نیازی به Welch نیست).

SC8

Question: Define full mediation, partial mediation, and competitive mediation.

EN: Full: indirect significant, direct (c') not significant. Partial: both indirect and direct significant (same sign). Competitive: indirect and direct significant with opposite signs, so total can be near zero.

FA: Full: اثر غیرمستقیم معنی‌دار، اثر مستقیم غیرمعنی‌دار. Partial: هر دو معنی‌دار (هم‌علامت). Competitive: اثر مستقیم و غیرمستقیم معنی‌دار ولی با علامت مخالف، پس total ممکن است نزدیک صفر شود.

SC9

Question: Are orthogonal contrasts independent? Give a two-layer answer.

EN: In coefficient space (math), yes: dot products are 0 so they partition variance without overlap. In data space, not always: if responses are correlated (like within-subject designs), test statistics can co-move.

FA: از نظر ضرایب (ریاضی)، بله: ضرب داخلی صفر است و همپوشانی ندارند. از نظر داده‌ها، همیشه نه: اگر پاسخ‌ها همبسته باشند (مثلا within-subject)، آماره‌ها می‌توانند با هم حرکت کنند.

SC10

Question: Three pairs show full agreement across methods, three pairs are method-sensitive. How do you report this?

EN: Report the stable pairs as strong conclusions (robust to assumptions). For the method-sensitive pairs, report them as cautionary and explain the disagreement (skew, different assumptions, different sensitivity).

FA: جفت‌های stable را با اطمینان گزارش کن چون در همه روش‌ها یکی هستند. جفت‌های method-sensitive را با احتیاط گزارش کن و بگو چرا ممکن است اختلاف باشد (skew، فرض‌های مختلف، حساسیت متفاوت).

SC11

Question: Why is omega-squared (omega^2) preferred over eta-squared (eta^2)?

EN: eta^2 is biased upward in small samples and does not penalize for the number of groups. omega^2 adjusts using MSE, so it is usually a less biased estimate of the population effect size.

FA: eta^2 در نمونه‌های کوچک معمولا خوش‌بینانه است و بابت تعداد گروه‌ها جریمه ندارد. omega^2 با MSE اصلاح می‌کند، پس معمولا bias کمتر دارد.

SC12

Question: When should you use FDR instead of Bonferroni or Holm?

EN: Use FDR when you have many tests and the goal is discovery (you can tolerate a small proportion of false positives). Use Bonferroni/Holm when you want strict familywise Type I control (confirmatory settings).

FA: وقتی تعداد تست زیاد است و هدف کشف است، FDR بهتر است چون توان بیشتری می‌دهد و فقط نسبت خطا را کنترل می‌کند. وقتی می‌خواهی خیلی سخت‌گیر باشی و FWER کنترل شود، Bonferroni یا Holm مناسب‌تر است.

Common Number Mistakes

توضیح فارسی: خطاهای عدددهی شایع

اینها مواردی هستند که دانشجوها معمولا در محاسبات یا نوشتن گزارش اشتباه می‌کنند:

  • If omega^2 computes to a negative number, always report it as 0 (truncate at zero). Never report negative effect size metrics.
  • For Tukey critical difference, make sure to use p (number of groups) to look up the q value, not df_error.
  • In mediation, the indirect-to-total ratio can be greater than 1. This is mathematically correct in competitive mediation. Don't assume it's an error.

Extra True/False Traps (From Other Exam Years)

توضیح فارسی: سوال‌های True/False از سال‌های دیگر

اینها سوال‌هایی است که تو سال‌های دیگر آمده و تو بخش اصلی نبود. بعضی‌ها خیلی ریز و دام‌دار هستند.

EQ1

Statement: A highly significant p-value (e.g., p = .001) is associated with a large effect size.

Answer: False.

Why (EN): P-value depends on sample size. A tiny effect with large n can give very small p. You need effect size measures separately.

چرا (FA): p-value به حجم نمونه بستگی دارد. اثر کوچک با n بزرگ می‌تواند p خیلی کوچک بدهد.

EQ2

Statement: For a given set of effects, the noncentrality parameter increases if error variance increases.

Answer: False.

Why (EN): lambda = n * sum(alpha_j^2) / sigma^2. If sigma^2 goes up, lambda goes down.

چرا (FA): اگر واریانس خطا زیاد شود، lambda کم می‌شود.

EQ3

Statement: If H0 is exactly true, the noncentrality parameter is exactly zero.

Answer: True. All alpha_j = 0, so lambda = 0.

FA: اگر H0 درست باشد، lambda = 0.

EQ4

Statement: If F = 1.0, you always fail to reject at 5% regardless of sample size.

Answer: True. The critical F at 5% is always > 1 for any df combination.

FA: مقدار بحرانی F در 5% همیشه بزرگ‌تر از 1 است.

EQ5

Statement: Omega squared changes if significance level goes from .05 to .01.

Answer: False. Omega^2 is computed from SS and df, not alpha.

FA: omega squared از SS و df حساب می‌شود، نه alpha.

EQ6

Statement: Contrasts [8, -2, -2, -2, -2] and [-.8944, .2236, .2236, .2236, .2236] give same p-values.

Answer: True. Second set is first scaled by a constant. Scaling does not change t or p.

FA: ضرب کنتراست در عدد ثابت، t و p را عوض نمی‌کند.

EQ7

Statement: Bootstrap CI bounds for indirect effect are symmetric about the estimate.

Answer: False. Bootstrap CIs can be asymmetric. Indirect effect distribution is often skewed.

FA: CI بوت‌استرپ نرمال فرض نمی‌کند و می‌تواند نامتقارن باشد.

EQ8

Statement: In one-way ANOVA, partial eta squared equals R-squared.

Answer: True. With one factor, partial eta^2 = SS_treat / (SS_treat + SS_error) = R^2.

FA: وقتی فقط یک عامل داریم، partial eta squared و R^2 یکی هستند.

EQ9

Statement: For both a priori and post hoc contrasts, FWER needs to be controlled.

Answer: False. A priori (planned) contrasts can use uncorrected alpha if few. Post hoc needs FWER control.

FA: a priori اگر کم و از قبل تعیین شده باشند لزوما اصلاح نمی‌خواهند.

EQ10

Statement: F sig at 5%. Using Tukey at 5%, at least one pair is significant.

Answer: True (with Tukey). Different from Q4 in main section (Bonferroni = False)! Tukey guarantees it.

FA: با Q4 بالا (Bonferroni) فرق دارد! Tukey تضمین می‌کند. اسم تست را دقیق بخون.

Tukey vs Bonferroni: exam trap

EN: "F sig + Tukey" = at least one pair sig (True). "F sig + Bonferroni" = not guaranteed (False).

FA: "F معنی‌دار + Tukey" = True. "F معنی‌دار + Bonferroni" = False.

EQ11

Statement: If omega^2 = 0, the F statistic must equal 1.

Answer: True. If no effect, E[MST] = E[MSE] so E[F] = 1.

FA: اگر اثری نباشد، E[F] = 1.

EQ12

Statement: Tukey, Bonferroni, and Scheffe all control FWER.

Answer: True. All three control FWER Type I, just with different methods.

FA: هر سه خطای خانواده‌ای نوع اول را کنترل می‌کنند.

EQ13

Statement: Power is higher at alpha = 0.10 than at alpha = 0.05.

Answer: True. Higher alpha = easier to reject = higher power.

FA: alpha بالاتر = power بیشتر.

Family-Wise Error Rate Formula (No Adjustment)

توضیح فارسی: محاسبه FWER بدون اصلاح

اگر t-test های جفتی بدون هیچ اصلاحی انجام شود، FWER چقدر می‌شود؟

The formula

m = p*(p-1)/2    (number of pairwise comparisons)
FWER = 1 - (1 - alpha)^m

Worked example (5 groups)

p = 5, alpha = 0.05, m = 10
FWER = 1 - (0.95)^10 = 1 - 0.5987 = 0.4013

EN: About 40% chance of false positive with no correction.

FA: حدود 40% احتمال false positive بدون اصلاح.

FW1

4 groups, alpha = 0.05. What is FWER with no adjustment?

EN: m = 6, FWER = 1 - (0.95)^6 = 0.265. About 26.5%.

FA: m = 6, FWER = 0.265. حدود 26.5%.

Reading Tukey Grouping Letters (SAS Output)

توضیح فارسی: خواندن حروف Tukey از خروجی SAS

حرف مشترک = فرق ندارند. هیچ حرف مشترکی = فرق دارند.

The rule

  • Shared letter = NOT significantly different
  • No shared letters = significantly different

Example from past exam

Tukey Grouping    Mean     N    Level
         A       9.700    10    2
         A
   B     A       9.200    10    4
   B
   C     B       8.100    10    5
   C
   C              6.600    10    3

         D       6.200    10    1
Level Letters

Pair decisions:

  • 2 vs 4: share "A" = NOT significant
  • 2 vs 5: no shared = significant
  • 2 vs 3: no shared = significant
  • 2 vs 1: no shared = significant
  • 4 vs 5: share "B" = NOT significant
  • 4 vs 3: no shared = significant
  • 4 vs 1: no shared = significant
  • 5 vs 3: share "C" = NOT significant
  • 5 vs 1: no shared = significant
  • 3 vs 1: no shared = significant

Common mistake

EN: Check ALL pairs, not just adjacent rows.

FA: همه جفت‌ها را چک کن نه فقط ردیف‌های مجاور.

Omega Squared from Full ANOVA Table

توضیح فارسی: omega squared از جدول کامل ANOVA

تو امتحان ستاره با F = 236.35 و p=4 و n=16 این فرمول لازم شد.

Original formula (with SS)

omega^2 = (SS_treat - df_treat * MSE) / (SS_total + MSE)

Shortcut (from F, p, n)

omega^2 = (p-1)(F-1) / ((p-1)(F-1) + n*p)   [n per group]
omega^2 = (p-1)(F-1) / ((p-1)(F-1) + N)      [N total]

Watch the denominator

EN: Two versions: n*p (n per group) or N (total). Check the exam.

FA: دقت کن n هر گروه یا N کل؟

Worked example (F = 236.35, p = 4, n = 16)

omega^2 = (3)(235.35) / ((3)(235.35) + 64)
        = 706.05 / 770.05 = 0.917

f = sqrt(0.917 / 0.083) = 3.32

EN: omega^2 = 0.917 (92% variance explained). f = 3.32. Both extremely large.

FA: omega^2 = 0.917. f = 3.32. هر دو خیلی بزرگ.

Mediation Types: Complete Classification

MT1

Statement: If a and b are both negative and c is positive, mediation is consistent.

Answer: True. a*b = positive (neg*neg). c = positive. Same direction = complementary/consistent.

FA: a*b مثبت. c مثبت. هم‌جهت = complementary/consistent.

Memory line

EN: Same sign of (a*b) and c' = complementary. Opposite signs = competitive.

FA: هم‌علامت = complementary. مخالف‌علامت = competitive.
