📚 Course Overview

🎯 Course Focus

This course covers advanced experimental design methods for behavioral sciences research. You'll learn to design, analyze, and interpret complex experiments using ANOVA, mediation analysis, and related techniques.

💻 Software Tools

We use SAS, R, and SPSS for statistical analysis. The emphasis is on understanding syntax and being able to run code line by line—not just clicking buttons.

📖 Key Textbook

Kirk, R. E. (2013). Experimental Design: Procedures for the Behavioral Sciences (4th ed.). SAGE Publications.

📝 Assessment

Weekly homework assignments, quizzes, and a final project. Focus on understanding concepts deeply, not just memorizing formulas.

📝 Effective Note-Taking Guide

Use this systematic approach when reading academic papers and textbooks:

1️⃣ First Pass (5 min)

  • Read title, abstract, introduction
  • Look at figures and tables
  • Read conclusion
  • Identify: What is the main question?

2️⃣ Second Pass (15 min)

  • Read carefully but skip proofs
  • Note key terms and definitions
  • Mark confusing parts for later
  • Identify: What methods were used?

3️⃣ Summarize

  • Purpose: What problem does this solve?
  • Method: How did they approach it?
  • Findings: What did they discover?
  • Contribution: Why does it matter?

4️⃣ Connect

  • How does this relate to other readings?
  • What are the limitations?
  • How can I apply this to my research?
  • What questions remain unanswered?

🇮🇷 راهنمای خلاصه‌نویسی دقیق

برای هر مقاله یا فصل کتاب این سؤالات رو جواب بده:

  1. مسئله چیه؟ نویسنده می‌خواد چه مشکلی رو حل کنه؟
  2. روش چیه؟ از چه تکنیک یا طراحی استفاده کرده؟
  3. نتایج کلیدی چین؟ مهم‌ترین یافته‌ها کدومان؟
  4. چرا مهمه؟ این کار چه کمکی به علم یا عمل می‌کنه؟
